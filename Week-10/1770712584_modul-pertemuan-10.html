FORM VALIDATION

Validasi 2 Layer: Client & Server

🎯 Capaian Pembelajaran

  • Kemampuan Akhir: Mahasiswa mampu menerapkan validasi pada formulir web menggunakan JavaScript dan PHP
  • Memahami pentingnya validasi 2 layer (client & server)
  • Mampu membuat validasi client-side dengan JavaScript
  • Mampu membuat validasi server-side dengan PHP
  • Mampu mengimplementasikan validasi pada operasi CRUD
  • Memahami teknik sanitasi dan escape data
💡 Catatan Penting: Contoh pada modul ini menggunakan Sistem Akademik sebagai studi kasus. Namun, konsep validasi ini dapat diterapkan pada project apapun yang kalian buat (Toko Online, Perpustakaan, Reservasi, dll). Sesuaikan aturan validasi dengan kebutuhan project masing-masing!

🛡 Mengapa Validasi Penting?

❌ Tanpa Validasi:
  • Data tidak konsisten
  • Rentan SQL Injection
  • Rentan XSS Attack
  • Database berisi data sampah
  • Aplikasi mudah crash
✅ Dengan Validasi:
  • Data terjamin valid
  • Aman dari serangan
  • User experience baik
  • Database bersih
  • Aplikasi stabil

📋 JENIS-JENIS VALIDASI

Berikut adalah jenis validasi yang umum digunakan di berbagai project:

1. Required (Wajib Diisi)
Field tidak boleh kosong. Contoh: Nama, Email, Password
2. Format/Pattern
Data harus sesuai format tertentu. Contoh: Email harus ada @, No HP harus angka
3. Length (Panjang)
Minimal/maksimal karakter. Contoh: Password min 6 karakter, Username max 20 karakter
4. Range (Rentang)
Nilai harus dalam rentang tertentu. Contoh: Umur 17-100, Harga > 0
5. Unique (Unik)
Tidak boleh duplikat di database. Contoh: Email, Username, NIM, Kode Produk
6. Match (Kecocokan)
Dua field harus sama. Contoh: Password & Konfirmasi Password
7. Type (Tipe Data)
Harus tipe data tertentu. Contoh: Angka, Tanggal, File gambar

Struktur Dasar Validasi JavaScript

JavaScript - Struktur Dasar // Ambil form element const form = document.getElementById('myForm'); // Event listener saat form di-submit form.addEventListener('submit', function(e) { // Ambil nilai input const nama = document.getElementById('nama').value.trim(); const email = document.getElementById('email').value.trim(); let errors = []; // Validasi: Required if (nama === '') { errors.push('Nama wajib diisi!'); } // Validasi: Format Email const emailPattern = /^[^\s@]+@[^\s@]+\.[^\s@]+$/; if (!emailPattern.test(email)) { errors.push('Format email tidak valid!'); } // Jika ada error, tampilkan dan batalkan submit if (errors.length > 0) { e.preventDefault(); // Batalkan submit alert(errors.join('\n')); return false; } // Jika tidak ada error, form akan ter-submit return true; });

Fungsi Validasi Reusable

Buat fungsi-fungsi validasi yang bisa digunakan ulang di berbagai form:

JavaScript - Fungsi Validasi Reusable // ======== FUNGSI VALIDASI REUSABLE ======== // Cek apakah field kosong function isEmpty(value) { return value.trim() === ''; } // Validasi format email function isValidEmail(email) { const pattern = /^[^\s@]+@[^\s@]+\.[^\s@]+$/; return pattern.test(email); } // Validasi panjang minimal function minLength(value, min) { return value.length >= min; } // Validasi panjang maksimal function maxLength(value, max) { return value.length <= max; } // Validasi hanya angka function isNumeric(value) { return /^\d+$/.test(value); } // Validasi nomor HP Indonesia function isValidPhone(phone) { return /^08[0-9]{8,11}$/.test(phone); } // Validasi angka dalam range function inRange(value, min, max) { const num = parseFloat(value); return num >= min && num <= max; }

Fungsi Validasi PHP Reusable

includes/validation.php <?php /** * Kumpulan Fungsi Validasi PHP * Bisa digunakan untuk berbagai project */ // Cek field kosong function required($value) { return !empty(trim($value)); } // Validasi email function validEmail($email) { return filter_var($email, FILTER_VALIDATE_EMAIL) !== false; } // Validasi panjang minimal function minLen($value, $min) { return strlen(trim($value)) >= $min; } // Validasi panjang maksimal function maxLen($value, $max) { return strlen(trim($value)) <= $max; } // Validasi angka function isNumber($value) { return is_numeric($value); } // Validasi angka dalam range function between($value, $min, $max) { return $value >= $min && $value <= $max; } // Validasi nomor HP Indonesia function validPhone($phone) { return preg_match('/^08[0-9]{8,11}$/', $phone); } // Sanitasi string (cegah XSS) function sanitize($value) { return htmlspecialchars(trim($value), ENT_QUOTES, 'UTF-8'); } ?>

Validasi Unique (Cek Duplikat di Database)

Validasi ini hanya bisa dilakukan di server karena memerlukan akses ke database:

PHP - Cek Data Duplikat <?php require_once 'config/database.php'; /** * Cek apakah nilai sudah ada di database * @param PDO $pdo - Koneksi database * @param string $table - Nama tabel * @param string $column - Nama kolom * @param mixed $value - Nilai yang dicek * @param mixed $exceptId - ID yang dikecualikan (untuk update) * @return bool */ function isUnique($pdo, $table, $column, $value, $exceptId = null) { if ($exceptId) { // Untuk UPDATE: kecualikan record yang sedang diedit $sql = "SELECT COUNT(*) FROM {$table} WHERE {$column} = ? AND id != ?"; $stmt = $pdo->prepare($sql); $stmt->execute([$value, $exceptId]); } else { // Untuk CREATE: cek semua record $sql = "SELECT COUNT(*) FROM {$table} WHERE {$column} = ?"; $stmt = $pdo->prepare($sql); $stmt->execute([$value]); } return $stmt->fetchColumn() == 0; } // ===== CONTOH PENGGUNAAN ===== $email = $_POST['email']; // Untuk CREATE (tambah data baru) if (!isUnique($pdo, 'users', 'email', $email)) { $errors[] = 'Email sudah terdaftar!'; } // Untuk UPDATE (edit data) $id = $_POST['id']; if (!isUnique($pdo, 'users', 'email', $email, $id)) { $errors[] = 'Email sudah digunakan user lain!'; } ?>
⚠ Penting untuk CRUD:
  • CREATE: Cek apakah data sudah ada (tanpa except)
  • UPDATE: Cek duplikat tapi kecualikan record yang sedang diedit
  • DELETE: Pastikan data ada sebelum dihapus

Validasi Real-time (Saat Mengetik)

JavaScript - Real-time Validation // Validasi real-time saat user mengetik/blur // Validasi saat input kehilangan fokus (blur) document.getElementById('email').addEventListener('blur', function() { const email = this.value.trim(); const feedback = this.nextElementSibling; if (isEmpty(email)) { setInvalid(this, 'Email wajib diisi'); } else if (!isValidEmail(email)) { setInvalid(this, 'Format email tidak valid'); } else { setValid(this); } }); // Validasi password saat mengetik (input event) document.getElementById('password').addEventListener('input', function() { const password = this.value; const strength = checkPasswordStrength(password); updateStrengthIndicator(strength); }); // Helper functions function setInvalid(input, message) { input.classList.remove('is-valid'); input.classList.add('is-invalid'); input.nextElementSibling.textContent = message; } function setValid(input) { input.classList.remove('is-invalid'); input.classList.add('is-valid'); } // Cek kekuatan password function checkPasswordStrength(password) { let strength = 0; if (password.length >= 8) strength++; if (/[a-z]/.test(password)) strength++; if (/[A-Z]/.test(password)) strength++; if (/[0-9]/.test(password)) strength++; if (/[^a-zA-Z0-9]/.test(password)) strength++; return strength; }
💡 Event yang Umum Digunakan:
  • blur - Saat input kehilangan fokus (recommended)
  • input - Setiap kali nilai berubah (real-time)
  • change - Saat nilai berubah dan blur
  • submit - Saat form di-submit

config/database.php <?php /** * Konfigurasi koneksi database menggunakan PDO * Singleton Pattern untuk efisiensi koneksi */ class Database { private static $instance = null; private $pdo; // Konfigurasi database private $host = 'localhost'; private $dbname = 'db_akademik'; private $username = 'root'; private $password = ''; private $charset = 'utf8mb4'; private function __construct() { try { $dsn = "mysql:host={$this->host};dbname={$this->dbname};charset={$this->charset}"; $options = [ PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION, PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC, PDO::ATTR_EMULATE_PREPARES => false ]; $this->pdo = new PDO($dsn, $this->username, $this->password, $options); } catch (PDOException $e) { die('Database connection failed: ' . $e->getMessage()); } } // Singleton: dapatkan instance koneksi public static function getInstance() { if (self::$instance === null) { self::$instance = new self(); } return self::$instance; } // Dapatkan PDO object public function getConnection() { return $this->pdo; } } // Helper function untuk akses cepat function db() { return Database::getInstance()->getConnection(); } ?>
Keuntungan Singleton Pattern:
  • Hanya 1 koneksi database yang dibuat
  • Menghemat resource server
  • Akses mudah dari mana saja dengan db()

5. Router - Entry Point

index.php (Entry Point) <?php /** * Entry Point - Simple Router * Semua request diproses melalui file ini */ session_start(); // Load Controller require_once 'controllers/MahasiswaController.php'; // Inisialisasi controller $controller = new MahasiswaController(); // Ambil action dari URL, default: index $action = $_GET['action'] ?? 'index'; $id = filter_input(INPUT_GET, 'id', FILTER_VALIDATE_INT); // Simple Routing switch ($action) { case 'index': $controller->index(); break; case 'create': $controller->create(); break; case 'store': $controller->store(); break; case 'show': $controller->show($id); break; case 'edit': $controller->edit($id); break; case 'update': $controller->update($id); break; case 'destroy': $controller->destroy($id); break; default: // Redirect ke index jika action tidak dikenali header('Location: ?action=index'); exit; } ?>
🎯 Alur Request:
  1. User akses index.php?action=create
  2. Router membaca parameter action=create
  3. Controller method create() dipanggil
  4. Controller memanggil View mahasiswa/create.php
  5. View dirender dalam Layout main.php

includes/validation.php <?php /** * Kumpulan fungsi validasi yang dapat digunakan ulang */ // Cek apakah nilai tidak kosong function required($value) { return !empty(trim($value)); } // Validasi format email function validEmail($email) { return filter_var($email, FILTER_VALIDATE_EMAIL) !== false; } // Validasi panjang minimum function minLen($value, $min) { return strlen(trim($value)) >= $min; } // Validasi panjang maksimum function maxLen($value, $max) { return strlen(trim($value)) <= $max; } // Validasi hanya angka function isNumber($value) { return is_numeric($value); } // Validasi nomor telepon Indonesia function validPhone($phone) { return preg_match('/^08[0-9]{8,11}$/', $phone); } // Sanitasi input untuk mencegah XSS function sanitize($value) { return htmlspecialchars(trim($value), ENT_QUOTES, 'UTF-8'); } // Cek keunikan data di database function isUnique($pdo, $table, $column, $value, $exceptId = null) { $sql = "SELECT COUNT(*) FROM {$table} WHERE {$column} = ?"; $params = [$value]; if ($exceptId) { $sql .= " AND id != ?"; $params[] = $exceptId; } $stmt = $pdo->prepare($sql); $stmt->execute($params); return $stmt->fetchColumn() == 0; } // Validasi hanya huruf dan spasi function alphaSpace($value) { return preg_match('/^[a-zA-Z\s]+$/', $value); } // Validasi alfanumerik function alphaNum($value) { return ctype_alnum($value); } ?>
