⚡ BAGIAN 1: PENGANTAR JAVASCRIPT

Apa itu JavaScript?

JavaScript adalah bahasa pemrograman yang berjalan di browser web. Berbeda dengan HTML (struktur) dan CSS (tampilan), JavaScript memberikan perilaku dan interaktivitas pada halaman web.

Peran JavaScript dalam Web Development:

  • Validasi Form: Memeriksa input sebelum dikirim ke server
  • Manipulasi DOM: Mengubah konten halaman secara dinamis
  • Event Handling: Merespons aksi pengguna (klik, hover, ketik)
  • Animasi: Membuat efek visual yang menarik
  • Komunikasi Server: Mengirim/menerima data tanpa reload (AJAX)

Cara Menambahkan JavaScript

Ada 3 cara untuk menambahkan JavaScript ke halaman HTML:

1 Inline JavaScript (Tidak Direkomendasikan)
<button onclick="alert('Halo!')">Klik Saya</button>
2 Internal JavaScript
<script> alert('Halo dari Internal JS!'); </script>
3 External JavaScript ✅ (Rekomendasi)
index.html <!-- Letakkan sebelum </body> --> <script src="js/script.js"></script>
js/script.js alert('Halo dari External JS!');
Best Practice: Selalu letakkan tag <script> sebelum </body> agar HTML selesai dimuat sebelum JavaScript dijalankan.

📦 BAGIAN 2: VARIABEL

Apa itu Variabel?

Variabel adalah "wadah" untuk menyimpan data. Seperti kotak berlabel yang bisa kita isi dengan nilai tertentu dan gunakan berkali-kali dalam program.

Analogi: Bayangkan variabel seperti laci berlabel di lemari. Laci "nama" berisi "Budi", laci "umur" berisi 20. Kita bisa buka laci tersebut kapan saja untuk mengambil atau mengubah isinya.

Mendeklarasikan Variabel

Di JavaScript modern, ada 3 cara mendeklarasikan variabel:

KeywordFungsiBisa Diubah?Rekomendasi
letVariabel yang nilainya bisa berubah✅ Ya⭐ Gunakan ini!
constKonstanta (nilai tetap)❌ Tidak⭐ Untuk nilai tetap
varCara lama (ES5)✅ Ya⚠ Hindari
Mendeklarasikan Variabel // Menggunakan let (nilai bisa berubah) let nama = "Budi Santoso"; let umur = 20; let ipk = 3.75; // Menggunakan const (nilai tetap) const NAMA_KAMPUS = "Universitas Tidar"; const MAX_SKS = 24; // Mengubah nilai variabel let umur = 21; // OK - let bisa diubah // NAMA_KAMPUS = "Kampus Lain"; // ERROR! const tidak bisa diubah

Aturan Penamaan Variabel

✅ Benar

  • namaLengkap (camelCase)
  • nama_lengkap (snake_case)
  • totalHarga
  • _private
  • $element

❌ Salah

  • 2nama (diawali angka)
  • nama-lengkap (pakai strip)
  • nama lengkap (pakai spasi)
  • let (reserved keyword)
  • class (reserved keyword)

⚙️ BAGIAN 3: FUNGSI (FUNCTION)

Apa itu Fungsi?

Fungsi adalah blok kode yang dirancang untuk melakukan tugas tertentu. Fungsi bisa dipanggil (dijalankan) kapan saja dan berapa kali pun diperlukan.

Analogi: Fungsi seperti resep masakan. Sekali ditulis, bisa dipakai berkali-kali untuk membuat masakan yang sama. Kita tidak perlu menulis ulang langkah-langkahnya setiap kali.

Koneksi dengan Flowchart

Ingat flowchart di Pertemuan 1? Fungsi adalah implementasi dari proses dan sub-proses dalam flowchart!

🔢 Flowchart

Mulai
↓
Input: nama, nilai
↓
Hitung rata-rata
↓
Tampilkan hasil
↓
Selesai

💻 Kode JavaScript

// Fungsi untuk proses
function hitungRataRata(nilai1, nilai2, nilai3) {
    let total = nilai1 + nilai2 + nilai3;
    let rata = total / 3;
    return rata;
}

// Memanggil fungsi
let hasil = hitungRataRata(85, 90, 78);
console.log(hasil); // 84.33

Sintaks Fungsi

Struktur Dasar Fungsi // 1. Function Declaration (cara klasik) function namaFungsi(parameter1, parameter2) { // Kode yang akan dijalankan return hasil; // Mengembalikan nilai (opsional) } // 2. Function Expression const namaFungsi = function(parameter) { return hasil; }; // 3. Arrow Function (ES6 - cara modern) const namaFungsi = (parameter) => { return hasil; }; // Arrow function singkat (satu baris) const kaliDua = (x) => x * 2;

js/script.js - Fungsi-fungsi Sistem Akademik // ======================================== // FUNGSI UNTUK SISTEM AKADEMIK // ======================================== // 1. Fungsi untuk menghitung IPK function hitungIPK(totalNilai, totalSKS) { if (totalSKS === 0) { return 0; } let ipk = totalNilai / totalSKS; return ipk.toFixed(2); // 2 angka desimal } // Contoh penggunaan let ipkMahasiswa = hitungIPK(288, 80); console.log(`IPK: ${ipkMahasiswa}`); // IPK: 3.60 // 2. Fungsi untuk format nama (capitalize) function formatNama(nama) { return nama .toLowerCase() .split(' ') .map(kata => kata.charAt(0).toUpperCase() + kata.slice(1)) .join(' '); } // Contoh penggunaan console.log(formatNama("AHMAD RIZKI")); // Ahmad Rizki console.log(formatNama("budi santoso")); // Budi Santoso

Fungsi Validasi Form (Sangat Penting!)

Validasi form adalah penggunaan JavaScript yang paling umum. Berikut fungsi-fungsi validasi untuk form login dan registrasi:

js/validasi.js - Fungsi Validasi Form // ======================================== // FUNGSI VALIDASI FORM // ======================================== // 1. Validasi field tidak boleh kosong function tidakKosong(nilai) { return nilai.trim() !== ""; } // 2. Validasi format email function validasiEmail(email) { const pattern = /^[^\s@]+@[^\s@]+\.[^\s@]+$/; return pattern.test(email); } // 3. Validasi panjang minimal password function validasiPassword(password) { return password.length >= 6; } // 4. Validasi konfirmasi password function passwordCocok(password, konfirmasi) { return password === konfirmasi; } // 5. Validasi NIM (hanya angka, 7-10 digit) function validasiNIM(nim) { const pattern = /^\d{7,10}$/; return pattern.test(nim); } // 6. Validasi nomor HP Indonesia function validasiHP(nomorHP) { const pattern = /^(08|62)\d{8,11}$/; return pattern.test(nomorHP); }
Contoh Penggunaan Fungsi Validasi // Test validasi console.log(tidakKosong("Ahmad")); // true console.log(tidakKosong(" ")); // false console.log(validasiEmail("[email protected]")); // true console.log(validasiEmail("ahmad@")); // false console.log(validasiPassword("123456")); // true console.log(validasiPassword("123")); // false console.log(validasiNIM("2024001")); // true console.log(validasiNIM("ABC123")); // false console.log(validasiHP("081234567890")); // true console.log(validasiHP("12345")); // false
Mengapa Validasi Penting? Validasi di JavaScript (client-side) memberikan feedback instan kepada user sebelum data dikirim ke server. Namun tetap harus ada validasi di PHP (server-side) untuk keamanan!

Script Validasi Login

js/login.js - Script Validasi Form Login // ======================================== // VALIDASI FORM LOGIN // ======================================== // Ambil elemen form const loginForm = document.getElementById('loginForm'); const emailInput = document.getElementById('email'); const passwordInput = document.getElementById('password'); const emailError = document.getElementById('emailError'); const passwordError = document.getElementById('passwordError'); // Fungsi untuk menampilkan error function tampilkanError(input, errorSpan, pesan) { input.classList.add('error'); input.classList.remove('success'); errorSpan.textContent = pesan; } // Fungsi untuk menghapus error function hapusError(input, errorSpan) { input.classList.remove('error'); input.classList.add('success'); errorSpan.textContent = ''; } // Event listener saat form di-submit loginForm.addEventListener('submit', function(event) { let isValid = true; // Validasi email if (!tidakKosong(emailInput.value)) { tampilkanError(emailInput, emailError, 'Email tidak boleh kosong!'); isValid = false; } else if (!validasiEmail(emailInput.value)) { tampilkanError(emailInput, emailError, 'Format email tidak valid!'); isValid = false; } else { hapusError(emailInput, emailError); } // Validasi password if (!tidakKosong(passwordInput.value)) { tampilkanError(passwordInput, passwordError, 'Password tidak boleh kosong!'); isValid = false; } else if (!validasiPassword(passwordInput.value)) { tampilkanError(passwordInput, passwordError, 'Password minimal 6 karakter!'); isValid = false; } else { hapusError(passwordInput, passwordError); } // Jika tidak valid, cegah submit if (!isValid) { event.preventDefault(); } });

📋 EVALUASI & TUGAS KELOMPOK

👥 Tugas Project Kelompok - Pertemuan 4

Setiap kelompok harus menambahkan JavaScript ke project yang sudah dibuat:

  1. js/validasi.js - Berisi fungsi-fungsi validasi (minimal 5 fungsi)
  2. js/login.js - Validasi form login (email & password)
  3. js/tambah.js - Validasi form tambah mahasiswa (semua field wajib diisi, NIM hanya angka, dll)
  4. js/script.js - Fungsi umum (hitungIPK, konversiNilai, formatNama, dll)
Ketentuan Tugas:
  • Minimal 5 fungsi validasi yang berbeda
  • Minimal 3 fungsi utilitas (seperti hitungIPK, formatNama, dll)
  • Setiap form harus memiliki validasi JavaScript
  • Tampilkan pesan error yang informatif kepada user
  • Gunakan console.log() untuk debugging
