🐘 BAGIAN 1: PENGANTAR PHP

Apa itu PHP?

PHP (PHP: Hypertext Preprocessor) adalah bahasa pemrograman server-side yang dirancang khusus untuk pengembangan web. Berbeda dengan JavaScript yang berjalan di browser (client), PHP berjalan di server.

Perbedaan Client-Side vs Server-Side:

💻 Client-Side (JavaScript)
  • Berjalan di browser user
  • Tidak bisa akses database
  • Untuk interaktivitas UI
  • Kode bisa dilihat user
🖥 Server-Side (PHP)
  • Berjalan di server
  • Bisa akses database
  • Untuk proses data
  • Kode tersembunyi

Sintaks Dasar PHP

Kode PHP ditulis di antara tag <?php dan ?>. PHP bisa disisipkan di dalam file HTML.

Sintaks Dasar PHP <?php // Ini adalah komentar satu baris /* Ini adalah komentar multi-baris */ // Menampilkan output echo "Halo, Dunia!"; // dengan echo print "Selamat Datang"; // dengan print // Setiap statement diakhiri dengan titik koma (;) ?>

Variabel di PHP

Di PHP, variabel dimulai dengan tanda $ (dollar). PHP adalah loosely typed, artinya tidak perlu mendeklarasikan tipe data.

🔶 JavaScript

let nama = "Ahmad"; let umur = 20; const IPK = 3.85; let aktif = true;

$nama = "Ahmad"; $umur = 20; $ipk = 3.85; $aktif = true;

🐘 PHP (Sekarang)

function hitungIPK($total, $sks) { if ($sks === 0) return 0; return number_format($total / $sks, 2); } function konversiNilai($nilai) { if ($nilai >= 85) return "A"; if ($nilai >= 70) return "B"; if ($nilai >= 55) return "C"; if ($nilai >= 40) return "D"; return "E"; }
functions.php - File Kumpulan Fungsi <?php // ============================================ // FUNGSI-FUNGSI UNTUK SISTEM AKADEMIK // ============================================ /** * Menghitung IPK dari total nilai dan SKS */ function hitungIPK($total_nilai, $total_sks) { if ($total_sks == 0) { return 0; } return number_format($total_nilai / $total_sks, 2); } /** * Konversi nilai angka ke huruf */ function konversiNilai($nilai) { if ($nilai >= 85) return "A"; if ($nilai >= 70) return "B"; if ($nilai >= 55) return "C"; if ($nilai >= 40) return "D"; return "E"; } /** * Format nama (Capitalize setiap kata) */ function formatNama($nama) { return ucwords(strtolower($nama)); } /** * Validasi email */ function validasiEmail($email) { return filter_var($email, FILTER_VALIDATE_EMAIL) !== false; } /** * Format tanggal Indonesia */ function formatTanggal($tanggal) { return date("d-m-Y", strtotime($tanggal)); } ?>
Cara Pakai: Gunakan require 'functions.php'; atau include 'functions.php'; di awal file PHP untuk menggunakan fungsi-fungsi tersebut.

🗃 BAGIAN 3: PENGANTAR MySQL

Apa itu MySQL?

MySQL adalah sistem manajemen database relasional (RDBMS) yang paling populer untuk aplikasi web. Database menyimpan data secara terstruktur dalam bentuk tabel.

Konsep Dasar Database:

  • Database: Kumpulan tabel yang saling berelasi (contoh: db_akademik)
  • Tabel: Struktur untuk menyimpan data sejenis (contoh: tabel mahasiswa)
  • Field/Kolom: Atribut data (contoh: nim, nama, email)
  • Record/Baris: Satu entri data lengkap
  • Primary Key: Field unik sebagai identitas record
  • Foreign Key: Field yang menghubungkan ke tabel lain

🔎 BAGIAN 4: QUERY DASAR (CRUD)

CRUD adalah singkatan dari Create, Read, Update, Delete - empat operasi dasar dalam database.

1. CREATE - Menambah Data (INSERT)

SQL - INSERT INTO -- Menambah satu data mahasiswa INSERT INTO mahasiswa (nim, nama, email, prodi, semester) VALUES ('2024001', 'Ahmad Rizki', '[email protected]', 'Teknologi Informasi', 4); -- Menambah beberapa data sekaligus INSERT INTO mahasiswa (nim, nama, email, prodi, semester) VALUES ('2024002', 'Budi Santoso', '[email protected]', 'Teknologi Informasi', 4), ('2024003', 'Citra Dewi', '[email protected]', 'Sistem Informasi', 2), ('2024004', 'Dina Putri', '[email protected]', 'Teknologi Informasi', 6);

2. READ - Membaca Data (SELECT)

SQL - SELECT -- Mengambil semua data SELECT * FROM mahasiswa; -- Mengambil kolom tertentu SELECT nim, nama, email FROM mahasiswa; -- Dengan kondisi WHERE SELECT * FROM mahasiswa WHERE prodi = 'Teknologi Informasi'; -- Dengan pengurutan ORDER BY SELECT * FROM mahasiswa ORDER BY nama ASC; -- Dengan pencarian LIKE SELECT * FROM mahasiswa WHERE nama LIKE '%Ahmad%'; -- Dengan batasan LIMIT SELECT * FROM mahasiswa LIMIT 10; -- Menghitung jumlah data SELECT COUNT(*) AS total FROM mahasiswa;
Halaman 10

3. UPDATE - Mengubah Data

SQL - UPDATE -- Mengubah satu field UPDATE mahasiswa SET semester = 5 WHERE nim = '2024001'; -- Mengubah beberapa field UPDATE mahasiswa SET email = '[email protected]', no_hp = '081234567890', alamat = 'Jl. Kapten Suparman No. 123, Magelang' WHERE nim = '2024001'; -- Update dengan kondisi UPDATE mahasiswa SET semester = semester + 1 WHERE prodi = 'Teknologi Informasi';

4. DELETE - Menghapus Data

SQL - DELETE -- Menghapus satu data DELETE FROM mahasiswa WHERE nim = '2024004'; -- Menghapus dengan kondisi DELETE FROM mahasiswa WHERE semester > 14; -- HATI-HATI! Ini menghapus SEMUA data DELETE FROM mahasiswa; -- Tanpa WHERE = hapus semua!
PENTING! Selalu gunakan WHERE pada perintah UPDATE dan DELETE! Tanpa WHERE, semua data akan terpengaruh. Backup data sebelum melakukan operasi berbahaya.

📋 EVALUASI & TUGAS KELOMPOK

👥 Tugas Project Kelompok - Pertemuan 5

Setiap kelompok harus menyelesaikan tugas berikut:

  1. Buat database db_akademik di phpMyAdmin
