🎯 Capaian Pembelajaran

💡 Apa itu Bootstrap?

Bootstrap adalah framework CSS open-source yang paling populer di dunia. Dikembangkan oleh Twitter, Bootstrap menyediakan komponen UI siap pakai dan sistem grid yang memudahkan pembuatan website responsif.

Halaman 1

📦 INSTALASI BOOTSTRAP

Ada beberapa cara untuk menggunakan Bootstrap dalam project. Cara yang paling mudah adalah menggunakan CDN (Content Delivery Network).

Metode 1: CDN (Recommended untuk Pemula)

Cukup tambahkan link CSS dan JavaScript Bootstrap di file HTML:

Template Dasar Bootstrap 5 <!DOCTYPE html> <html lang="id"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1"> <title>My Bootstrap App</title> <!-- Bootstrap CSS --> <link href="https://cdn.jsdelivr.net/npm/[email protected]/dist/css/bootstrap.min.css" rel="stylesheet"> </head> <body> <!-- Konten halaman --> <h1>Hello, Bootstrap!</h1> <!-- Bootstrap JS (taruh sebelum </body>) --> <script src="https://cdn.jsdelivr.net/npm/[email protected]/dist/js/bootstrap.bundle.min.js"></script> </body> </html>
Penting! Meta tag viewport WAJIB ada agar Bootstrap responsif bekerja dengan benar di perangkat mobile.

Metode 2: Download File

Download file Bootstrap dari getbootstrap.com dan simpan di folder project:

Struktur Folder dengan Bootstrap Lokal sistem-akademik/ ├── css/ │ ├── bootstrap.min.css # File Bootstrap │ └── style.css # Custom CSS kita ├── js/ │ ├── bootstrap.bundle.min.js │ └── script.js ├── index.php └── ...
Tips: Untuk praktikum ini, gunakan CDN agar lebih mudah. Untuk production, download file lokal agar tidak bergantung pada koneksi internet.

Class Tabel Bootstrap:

🔗 INTEGRASI DENGAN PROJECT

Sebelum vs Sesudah Bootstrap

❌ Sebelum (CSS Manual)

  • Menulis CSS dari nol
  • Responsive manual
  • Desain tidak konsisten
  • Banyak kode repetitif

✅ Sesudah (Bootstrap)

  • Komponen siap pakai
  • Auto responsive
  • Desain profesional
  • Kode lebih ringkas

Contoh: Dashboard dengan Bootstrap

dashboard.php dengan Bootstrap <?php require_once 'includes/auth.php'; cekLogin(); require_once 'config/database.php'; $totalMhs = $pdo->query("SELECT COUNT(*) FROM mahasiswa")->fetchColumn(); ?> <!DOCTYPE html> <html lang="id"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1"> <title>Dashboard</title> <link href="https://cdn.jsdelivr.net/npm/[email protected]/dist/css/bootstrap.min.css" rel="stylesheet"> </head> <body> <!-- Navbar --> <nav class="navbar navbar-expand-lg navbar-dark bg-primary"> <div class="container"> <a class="navbar-brand" href="#">🎓 Sistem Akademik</a> <!-- ... menu items ... --> </div> </nav> <!-- Main Content --> <div class="container mt-4"> <h2 class="mb-4">Selamat Datang, <?= getUserNama() ?>!</h2> <!-- Stats Cards --> <div class="row"> <div class="col-md-4 mb-3"> <div class="card text-white bg-primary"> <div class="card-body"> <h5 class="card-title">Total Mahasiswa</h5> <p class="display-4"><?= $totalMhs ?></p> </div> </div> </div> </div> </div> <script src="https://cdn.jsdelivr.net/npm/[email protected]/dist/js/bootstrap.bundle.min.js"></script> </body> </html>

📋 TUGAS PRAKTIKUM

🎨 Tugas: Upgrade Project dengan Bootstrap

Ubah tampilan project Sistem Akademik yang sudah dibuat menggunakan Bootstrap 5:

  1. Tambahkan Bootstrap CSS & JS via CDN
  2. Ubah Navbar menggunakan komponen Bootstrap
  3. Ubah Dashboard menggunakan Cards Bootstrap
  4. Ubah Form Login dengan Bootstrap Form
  5. Ubah Tabel Mahasiswa dengan Bootstrap Table
  6. Ubah Form Tambah/Edit menggunakan Grid Layout
  7. Tambahkan Alert untuk pesan sukses/error
  8. Tambahkan Modal untuk konfirmasi hapus

Kriteria Penilaian

AspekBobotKriteria
Implementasi Bootstrap25%CDN terpasang dengan benar
Navbar Responsif15%Menu collapse di mobile
Grid System20%Layout responsive di semua halaman
Komponen Bootstrap25%Cards, Tables, Forms, Alerts
Konsistensi Design15%Tampilan seragam & profesional
Catatan Penting:
  • Pastikan meta viewport ada di semua halaman
  • Test di berbagai ukuran layar (HP, Tablet, Desktop)
  • Boleh menambahkan custom CSS untuk penyesuaian
  • Dokumentasikan perubahan yang dilakukan
